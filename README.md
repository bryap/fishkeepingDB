# Project Abstract: Something’s Fishy(DB)
## Overview
Something’s Fishy(DB) is a comprehensive relational database system designed to bridge the gap between ecological hobbyist needs and commercial aquarium management. While the fishkeeping industry relies on the precise interplay of biological, environmental, and logistical data, hobbyists often lack a centralized tool to manage these variables. This project delivers a scalable, highly normalized architecture capable of tracking everything from delicate aquatic biomes to complex retail inventory.
